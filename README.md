# Proyecto-Final


Este proyecto es un sistema CRUD completo para la gestión de estudiantes, desarrollado con **Spring Boot**, **Java**, **JUnit**, **Mockito**, y un frontend moderno basado en **SB Admin 2**.

El sistema permite crear, listar, actualizar, buscar y eliminar estudiantes mediante una API REST.
